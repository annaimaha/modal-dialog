# Modal Dialog using reactjs version 16.2.0

### Basic Requirements
Node Js version > 6

## Install
1. Make sure you have a recent version of Node.js installed.
2. Clone or download the file.
3. *npm install* 
4. *npm start*

## To Change the theme or color
